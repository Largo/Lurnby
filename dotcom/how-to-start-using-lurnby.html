I've never been more excited for anything. This has everything I want!

Lurnby is still in progress. What this means is that we fibbed a little bit. Not all of the features we mentioned earlier are live already. Sorry.

What works now?

  1. Add content from the web, we even have chrome and firefox extensions.
  2. Add your ebooks if they are in epub format.
  3. Add content via email.
  4. You can set your personal reading preferences, and you can read without ads.
  5. You can highlight as you read and you can create groups for organizing your highlights.
  6. Export your highlights + notes in txt/json format
  7. Pdf support - partly, text and images are processed fine.
  8. Spaced repetition practice for your highlights.

What's coming?

  1. Active Recall Practice
  2. WYSIWYG editor for document creation
  3. Search
